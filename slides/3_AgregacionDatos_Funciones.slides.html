3.1 Cadenas de caracteres¶

Indexación¶

  • El primer carcter de una cadena ocupa la posicin 0
  • str[i] $\rightarrow$ carcter situado en la posicin i de la cadena de caracteres str
  • str[i:j] $\rightarrow$ sub-cadena formada por los caracteres situados entre las posiciones i y j-1 de str, ambas incluidas
  • str[i:j:k] $\rightarrow$ anlogo a str[i:j], utilizando paso k
  • Se pueden utilizar ndices negativos: -1 corresponde al ltimo carcter de la cadena

 Módulos¶

  • Un mdulo es un archivo que contiene un conjunto de definiciones de funciones y/o v variables
  • Para utilizar las funciones y las variables de un mdulo en otros programas hay que importarlas.
  • Formas de importar un mdulo genrico, MOD1:
# a. Importar el mdulo y acceder a l con el nombre mm
import MOD1 as mm
# Utilizar una funcin del mdulo importado
mm.funcin_del_mdulo()

# b. Importar algunas funciones del mdulo para utilizarlas directamente
from nombre_del_mdulo import funcin_1[,funcin_2,...,funcin_n]) 
# Utilizar  las funciones importadas
funcin_1()

# c. Importar todas las funciones del mdulo para utilizarlas directamente
from nombre_del_mdulo import *
# Utilizar las funciones importadas
funcin_1()
funcin_2()
...
funcin_n()
